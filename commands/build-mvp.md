# /build-mvp

Design a validation-focused MVP.

Analyze:

* Core hypothesis
* Target users
* Key workflows
* Essential features
* Success metrics

Output:

1. MVP Goal
2. Core Features
3. Features To Exclude
4. Success Metrics
5. Validation Plan
6. Development Priorities

Optimize for learning speed rather than feature completeness.

Goal:

Build the smallest product capable of validating the startup's core assumption.
